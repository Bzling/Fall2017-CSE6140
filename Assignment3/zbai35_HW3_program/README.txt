Open the command line window in this folder.

To get the results of dynammic programming:
Input: javac DynamicProgamming.java
       java DynamicProgramming <datafile_name> <output_file_name>
       (For example:
       java DynamicProgramming 1000.txt zbai35_output_dp_1000.txt)

To get the results of Divide and Conquer:
Input: javac DivideConquer.java
       java DivideConquer <datafile_name> <output_file_name>
       (For example:
       java DivideConquer 1000.txt zbai35_output_dc_1000.txt)
